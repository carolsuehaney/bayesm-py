# bayesm-py
Experimental Python/C++ port of bayesm in R (see CRAN https://cran.r-project.org/web/packages/bayesm/index.html), using the original R package that was created by Peter Rossi [aut, cre], Robert McCulloch [ctb], Wayne Taylor [ctb], Dan Yavorsky [ctb].  Contains many  models used in marketing and other applications.  Code quality stable, statistical validation and documentation still in progress.
